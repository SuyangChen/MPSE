### Description of 3rd party packages ###

VisualHull:
https://github.com/maximm8/VisualHull
Matlab implementation of basic visual hull computation from silhoutes

ShadowArtDemo:
https://graphics.stanford.edu/~niloy/research/shadowArt/shadowArt_sigA_09.html
Demo of Shadow Art algorithm.
Use 'wine start ShadowArtSetup.msi' to install Windows application on Linux.
